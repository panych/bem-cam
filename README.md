Простой шаблон проекта (шапка, контент, футер) для работы с bem tools
========================

    Очень простые примеры создания своих блоков в bemhtml. Работа с bem-bl библиотекой.

* Cкачать проект.
* Установить зависимости.

        npm install

* Запустить бэм сервер

        bem server
        
* Или мейк

        bem make
